#  Inventory & Billing System

---

## Features

- **Product Management:** Add/edit/delete products; automatic barcode generation and images for labels.
- **Supplier Management:** Track supplier info, project details, purchased products, and budgets.
- **Billing:** Create customer bills, auto-fill with barcode scanner (like POS systems). View, print, and download bills.
- **Barcode Scanning:** Scan product barcodes with phone or desktop camera to add items to bills instantly.
- **Product Labels:** Generate and print barcode labels with company logo, product info, and price.
- **Reports:** Top-selling products, sales stats, and CSV export.
- **Print/Download:** PDF bills and CSV exports for products, suppliers, and transactions.
- **Mobile-friendly:** Responsive UI, installable as a PWA (Progressive Web App) for easy use on mobile.
